# PHP md5碰撞
### 类型I
```php
$str1 = $_GET['str1'];
$str2 = $_GET['str2'];
if (md5($str1) == md5($str2)) {
    echo 'ok!';
}
```
* 在`php`中，`==`是弱类型比较，比较值，并不比较类型，因此若`str1`和`str2`的值相等即可，似乎是`MD5`碰撞
* 实则不然，若计算的`md5`结果为`0e830400451993494058024219903391`这样的结果，即字母`e`开头，会被`php`解析为科学计数法
* 由于`0`的任意非`0`次方均为`0`,所以实现了一种伪碰撞。
